# Spam Classifier ML Project

An end-to-end NLP spam classifier built with Python and Scikit-learn.

## Features

- Text preprocessing
- TF-IDF vectorization
- Logistic Regression model
- Naive Bayes comparison
- Confusion matrix analysis
- False positive / false negative inspection
- Probability prediction
- Threshold tuning
- CLI prediction tool

---

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
